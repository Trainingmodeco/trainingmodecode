# 04 · CARDIO MODE

--- PROMPT 1: Cardio setup ---

Build Cardio setup: mode pills (OUTDOOR RUN / TREADMILL / HIIT-TABATA), distance-or-time goal picker, pace target auto-set from the chosen distance/goal (editable), voice-coach note ("uses your Audio settings"). Bonus-XP note: XP scales with distance/effort. START (gold).

--- PROMPT 2: Live run screen ---

Live outdoor run: the cardio ring timer art (/assets/rings/ring-cardio.png, ~400–480px, transparent PNG) with elapsed time large in the center (Orbitron 900) and a progress count-line around the ring. Big metric row: PACE (primary, large) · DISTANCE · TIME. GPS route map card below (dark map, violet route line). Pace-vs-target bar: green in range, orange slow, red fast. Voice coach fires "3-2-1 start", encouragement, "pick up the pace" when drifting below target, "ease up" when way over. Pause/end controls. Treadmill mode = same screen minus GPS map (time/interval based).

--- PROMPT 3: Interval / Tabata live ---

HIIT/Tabata live screen: phase banner "▲ WORK" (orange) / "▼ REST" (blue) with the ring timer showing the phase countdown ("10s on" style), round x/y, big phase seconds, auto-advancing with beeps, session progress bar. Config from setup (work secs / rest secs / rounds).

BEHAVIOR: cardio can be launched standalone OR appended from Workout Builder / Quick Mission / Combat Conditioning ("add cardio"); when appended it auto-prompts right after the main workout ends and grants bonus XP on completion.
