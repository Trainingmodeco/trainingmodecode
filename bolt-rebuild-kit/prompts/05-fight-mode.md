# 05 · FIGHT MODE (Fight Focus · Combo Coach · Practice Mode)

--- PROMPT 1: Fight Mode landing ---

FIGHT MODE landing: banner header, cards with one-line descriptions: FIGHT FOCUS ("Round timer — fight-paced work/rest rounds"), COMBO COACH ("Called combos at cadence — hands up"), PRACTICE MODE ("Learn every strike, step by step"), plus COMBAT CONDITIONING cross-listed (goes to the same feature as Fit Mode's).

--- PROMPT 2: Fight Focus ---

Round timer with two distinct states, using /assets/rings/ring-fight.png (~394px):
- WORK: red/violet ring, round countdown large, "ROUND x OF y", glanceable header of total elapsed + rounds left. Last-10-seconds cue: ring flashes + beep. Round-end bell → auto 3-2-1 → rest.
- REST: same ring hue-shifted blue, rest countdown, "NEXT: ROUND x", auto-advances back to work — user never touches controls mid-session.
- Setup: rounds, round length, rest length presets (fight-style 3:00/1:00 default).

--- PROMPT 3: Combo Coach ---

Cadence combo caller using its own ring art:
- Current combo displayed LARGE on screen as the call-out (e.g. "1 · 2 · SLIP · 3"), with the NEXT combo previewed small underneath so the fighter can anticipate.
- Pulsing orb/ring whose pulse speed = cadence (faster cadence, faster throb — tempo is felt, not read).
- Rush Mode finish: final 30s the ring shifts orange and cadence speeds up.
- Optional combo-streak counter (combos completed without pause) for a light score hook.
- Setup: discipline, cadence, session length, voice on/off (auto voice, no manual voice-coach needed).

--- PROMPT 4: Practice Mode ---

Technique library:
- Discipline continue-card using the gender-matched framed banner art (/assets/practice/{discipline}-{gender}.png — fighter on the right, text overlays the built-in left space): "CONTINUE LEARNING / LESSON n · NAME / ▶ RESUME".
- Lesson list per discipline: strike rows (Jab, Cross, Hook, Uppercut, kicks, knees, elbows by discipline) with status (done ✓ / current / locked), each opening a lesson page with a video-tutorial slot, key form cues, and "DRILL IT".
- Drill It → shadowbox drill screen: called reps of that strike at cadence + multi-strike combo call-outs mixing learned strikes.
- Category tabs include Strikes / Defense / Footwork with content rows for each.
